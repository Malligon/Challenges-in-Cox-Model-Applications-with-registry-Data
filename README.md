# Challenges-in-Cox-Model-Applications-with-registry-Data
R codes used to create the "Challenges in Cox Model Applications with registry Data" article, with data generation, biased and unbiased estimates, and figures to compare results. These codes can be reproduced on other data sets, and are meant to help researchers in their survival analysis.

Contains 7 main files, each one linked to the dedicated part in the paper:
  - `Code - 01 - Cox model applied to right-censored data.R`
  - `Code - 02 - Cox model applied to more complex data - troncature.R`
  - `Code - 03 - Cox model applied to more complex data - competing risks.R`
  - `Code - 04 - Cox model applied to more complex data - recurrent event.R`
  - `Code - 05 - Cox model with interactions.R`
  - `Code - 06 - Stratified Cox model and sub-analysis.R`
  - `Code - 07 - Cox model with time-dependent effect.R`
